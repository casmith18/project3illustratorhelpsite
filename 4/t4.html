Tutorial 4: Working with Color

Why This Matters

Color sets the tone of your design. Whether you’re going for something clean and modern or bold and energetic, the right color choices make a huge difference. In this tutorial, we’ll pick a color palette and apply it to your layout. This will help guide the style of the entire spread and make it look more polished.

Step 1: Pick a Color Using the Color Picker

Select one of your layout shapes. In the Properties panel on the right, you’ll see two boxes—one for Fill and one for Stroke. Click the Fill box to open the color picker.

You can choose a color from the spectrum or type in a specific value. If you have a brand color or a specific tone in mind, try using the HEX code field to paste it in directly.

Try setting your photo box to a bold feature color like navy blue or burnt orange. This can act as a background if you don’t end up using a photo.

Screenshot suggestion: Color picker open with a Fill color selected and HEX input visible.

Step 2: Use Swatches for Quick Color Choices

If you want to explore preset colors, open the Swatches panel by going to Window > Swatches. This panel gives you quick access to themed color sets and gradients.

Pick a few colors that look good together. You can apply one to your background shape, another to a header box, and a third for accent elements. Keep it simple and consistent.

Screenshot suggestion: Swatches panel open with a few colors applied to layout blocks.

Step 3: Create and Save a Custom Color Group

Once you’ve found a few colors you like, save them as a group so you can use them later. In the Swatches panel, click the folder icon to create a new color group. Name it something like “Spread Colors” or “Layout Theme.”

You can drag any color swatch into this group to keep everything organized.

Screenshot suggestion: Custom color group being created and named inside the Swatches panel.

Step 4: Understand CMYK vs RGB

Since this is a print layout, you should be using CMYK color mode. That gives you a more accurate idea of how things will look on paper. If you’re not sure what mode your document is in, go to File > Document Color Mode. Make sure CMYK is checked.

If this was a design meant only for screens, you’d use RGB instead.

Screenshot suggestion: File > Document Color Mode menu with CMYK selected.

Next Steps

Your layout is now starting to look intentional. With the structure and color in place, it’s time to add actual content. In the next tutorial, we’ll focus on adding and styling the text that gives your magazine spread its message.
