Tutorial 1: Setting Up Your Workspace

What We’re Making

In this series, we’ll be creating a clean and modern magazine spread using Adobe Illustrator. You might be wondering why we’re using Illustrator instead of something like InDesign. InDesign is great for handling long documents with lots of pages, but Illustrator gives you way more control when it comes to designing custom layouts and unique visuals. If you’re working on a two-page feature with heavy design elements, Illustrator is the better tool. Think of it like using Photoshop for photos and Illustrator for layout-driven art pieces.

This tutorial will show you how to start your project the right way. You’ll set up your document, adjust your artboards, and save a custom preset so next time is even easier.

Step 1: Open Illustrator and Start a New Document

Go ahead and open Adobe Illustrator. Once it’s loaded, click “New File” on the welcome screen.

In the new document window, type in your dimensions. Since we’re creating a two-page magazine spread, set the width to 17 inches and the height to 11 inches. This gives us the space for two side-by-side pages. Make sure the orientation is set to Landscape. Choose “Print” as the profile and “CMYK” as the color mode if you’re planning to send it to a printer. If this is just for screens, you can use “RGB” instead.

Once everything is set, hit “Create.”

Screenshot suggestion: New Document window with 17 in x 11 in dimensions filled out and CMYK selected.

Step 2: Understand Artboards and Units

You’ll now see a blank canvas. This is your artboard. It works like your digital sheet of paper. Since we’re making a two-page spread, this single wide artboard will let us treat both pages as one continuous layout.

If you prefer different units (like pixels or millimeters), you can change them in the Properties panel or by right-clicking the ruler. Inches are usually best for print, so let’s stick with that for now.

Screenshot suggestion: Artboard with rulers visible and dimensions labeled clearly.

Step 3: Save a Preset for Future Projects

To avoid typing all this in again later, save it as a custom preset. Click “New File” again, enter your settings like before, and look for a small icon that lets you save the preset. Give it a name like “Magazine Spread” and click Save. Next time you’ll be able to start faster.

Screenshot suggestion: Preset being saved with a clear name entered in the dialog box.

Next Steps

You’ve now got your space ready. Up next, we’ll start adding some shapes to build the base layout of our spread. This is where things really start to take form.
